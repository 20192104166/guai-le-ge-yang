# 拐了个羊（Guai Le Ge Yang）

> 一款基于蒙古族传统民俗游戏「沙嘎」（羊拐 / 嘎拉哈）的抖音小游戏。
> 集弹、抛、抓三种玩法于一体，传播草原文化。

## 🎮 玩法

| 玩法 | 类型 | 说明 |
|---|---|---|
| **弹沙嘎** | 技巧型对战 | 弹射击中同形态羊拐赢取，含 ELO 段位天梯 |
| **抛沙嘎** | 运气型对赌 | 抛 4 枚按 35 种组合计分，"撂四样"为吉祥组合 |
| **抓沙嘎** | 反应型挑战 | 沙包腾空时摆放羊拐成指定形态，关卡模式 |

## 🛠 技术栈

- **客户端**：Unity 2022 LTS（2D 模式）+ C#
- **后端**：抖音云开发（云函数 + 云数据库 + 云存储）
- **物理**：Unity Box2D + 加权随机混合方案
- **平台**：抖音小游戏
