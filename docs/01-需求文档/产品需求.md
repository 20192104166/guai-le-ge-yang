# 产品需求文档（PRD）

> 版本：v0.3（**所有关键决策已确认**，Phase 0 决策阶段完成）
> 最后更新：2026-05-15

---

## 1. 项目背景

「拐了个羊」是一款基于蒙古族传统民俗游戏「沙嘎」（羊拐 / 嘎拉哈）改编的抖音小游戏。沙嘎是利用羊后蹄踝骨的四个面进行的游戏，每个面对应草原上的动物，富有民族特色与文化趣味。

本项目目标是：在抖音小游戏平台上，以轻量化、休闲化的方式还原沙嘎游戏，传播少数民族传统文化。

---

## 2. 目标用户

- **平台**：抖音小游戏（用户基础广泛，偏年轻、偏休闲）
- **目标人群**：
  - 休闲游戏玩家
  - 对中国传统文化、少数民族民俗有兴趣的用户
  - 怀旧人群（老一辈的童年游戏）
  - 喜欢竞技 + 收集的玩家（PvP + 图鉴）

---

## 3. 核心玩法对象：羊拐（沙嘎）

羊拐是连接动物后蹄和小腿的踝骨，有四个明显不同的面：

| 面型 | 形态特征 | 对应动物 | 概率参考 |
|---|---|---|---|
| 宽凸面（S）| 较宽、外凸 | 绵羊 | ~ 35% |
| 宽凹面（G）| 较宽、内凹 | 山羊（或牛） | ~ 30% |
| 窄凸面（H）| 较窄、外凸 | 骏马 | ~ 18% |
| 窄凹面（C）| 较窄、内凹 | 骆驼 | ~ 12% |

详细物理参数见 [`../06-游戏设计/羊拐物理设计.md`](../06-游戏设计/羊拐物理设计.md)。

---

## 4. 三个核心玩法

三种玩法**各自独立的关卡/模式**，主菜单可任选其一进入。

### 4.1 弹沙嘎（精准对决 · 技巧型）
- **玩法**：弹射己方羊拐击中同形态的对方羊拐
- **胜负**：一方羊拐输完即败
- **模式**：单机 AI / PvP 在线 / 异步好友 PK
- 详见 [`../06-游戏设计/玩法-弹沙嘎.md`](../06-游戏设计/玩法-弹沙嘎.md)

### 4.2 抛沙嘎 / 撂四样（运气比拼 · 运气型）
- **玩法**：抛 4 枚羊拐，按组合计分
- **胜负**：先达目标分 / 对赌赢光对方羊拐
- **模式**：单机 / PvP / 异步
- 详见 [`../06-游戏设计/玩法-抛沙嘎.md`](../06-游戏设计/玩法-抛沙嘎.md)

### 4.3 抓沙嘎（手眼协调 · 挑战型）
- **玩法**：沙包腾空时摆放羊拐成指定形态
- **胜负**：纯单人挑战，不设对手；按通关数 / 用时上排行榜
- **模式**：单机关卡模式 + 异步好友比拼
- 详见 [`../06-游戏设计/玩法-抓沙嘎.md`](../06-游戏设计/玩法-抓沙嘎.md)

## 5. 美术与 UI 方向

- **整体风格**：**Q版可爱蒙古风** ✅
- **保留元素**：蒙古包、草原、毡毯、马头琴、传统纹样（盘长纹/云纹/八宝纹）✅
- **音乐**：传统蒙古音乐（先以马头琴风格试做）✅

详细规范见 [`../03-设计规范/美术风格指南.md`](../03-设计规范/美术风格指南.md)。

---

## 6. 商业化模式

- **PvP 模式**：纯免费，不插入广告（保证竞技公平 + 体验流畅）
- **单机模式**：
  - 失败后看广告复活
  - 关键回合"反悔"重抛看广告
  - 双倍奖励看广告
- **后期可选**：内购皮肤 / 图鉴解锁包（不影响平衡性）

---

## 7. 社交系统

| 模块 | 说明 |
|---|---|
| **排行榜** | 三玩法分榜（弹用 ELO / 抛比单局高分 / 抓比通关进度）+ 综合大师榜 + 好友榜 |
| **邀请 PK** | 抖音好友互相邀请对战 |
| **图鉴收集** | 沙嘎样式图鉴（不同地区款式）+ 蒙古文化卡片（待定） |

详细机制见 [`../06-游戏设计/玩法总览.md`](../06-游戏设计/玩法总览.md) 中的"社交与排行榜"章节。

## 8. 技术挑战

| 挑战 | 描述 | 状态 |
|---|---|---|
| Unity 抖音小游戏包体控制 | Unity WebGL/Tiny 输出在抖音平台的体积、性能 | Phase 1 必须验证 |
| 不规则物体 2D 物理仿真 | 用 2D 物理 + 加权随机模拟真实落地分布 | Phase 1 重点 |
| 后端服务（PvP / 异步 / 排行榜） | 需要服务端 + 数据库 + 实时通讯 | Phase 1.5 预研 |
| 多种交互手势 | 弹、抛、抓三种差异较大的手势 | Phase 2~4 逐一实现 |

## 10. 已确认决策详情

### 10.1 后端方案：抖音云开发
- 云函数 + 云数据库 + 云存储 + 抖音账号一键登录
- 详细架构见 [`../02-技术文档/技术选型.md`](../02-技术文档/技术选型.md) §3 + 后续即将编写的 `服务端架构.md`

### 10.2 PvP 形式：V1.0 仅异步
- 玩家 A 完成一局录像 → 上传云端 → 玩家 B 下载回放并应战 → 双方结果合并结算
- 不需要实时房间匹配、不需要 WebSocket 长连接（V1.0）
- 实时 PvP 视 V1.0 上线后用户量与反馈再决定

### 10.3 抛沙嘎计分表 v1.0
- 见 [`../06-游戏设计/玩法-抛沙嘎.md`](../06-游戏设计/玩法-抛沙嘎.md) §3 完整 35 种组合
- 设计原则：越稀有越高分 + 吉祥组合给文化加成
- 后续可在 Phase 2 实际测试时微调数值

### 10.4 单局时长机制
- 弹/抛沙嘎：按胜负结算（无固定时长上限）
- 抓沙嘎：关卡模式（单局 = 5 关挑战）
- 软兜底：单回合 30s 未操作自动出手；单局 5 分钟弹提示；10 分钟强制结算

### 10.5 排行榜机制
- 弹沙嘎天梯榜：ELO 段位（6 级蒙古主题段位）
- 抛沙嘎手气榜：周榜单局最高分
- 抓沙嘎挑战榜：永久最高关卡 + 周榜通关速度
- 沙嘎大师榜：三玩法加权综合
- 好友榜：与抖音好友数据打通

### 10.6 图鉴系统：双轨
- **羊拐皮肤图鉴**：按地区款式（内蒙古/外蒙古/青海/新疆/藏族/达斡尔 等）
- **蒙古文化卡片图鉴**：动物、乐器（马头琴/托布秀尔/四胡）、纹样、节日（那达慕/白月）、传说人物等
- 解锁方式：自然游戏解锁 + 成就解锁 + 看广告抽卡 + 节日活动

## 11. 范围声明

### 11.1 MVP 包含
- 三个核心玩法可玩（单机 AI 模式）
- Q 版蒙古风美术（最小可发布资源量）
- 物理仿真（2D + 加权随机）
- 基础 UI / 主菜单 / 设置
- 后端预留接口（先不接 PvP）

### 11.2 V1.0 包含（首发版本）
- MVP 全部
- ✅ PvP 异步对战
- ✅ 三玩法排行榜 + 综合大师榜 + 好友榜
- ✅ 图鉴收集（羊拐皮肤 + 文化卡片）
- 激励视频广告（单机模式）

### 11.3 V1.1+ 迭代
- 实时 PvP（视 V1.0 反响决定）
- 内购系统
- 节日活动 / 新玩法变体
