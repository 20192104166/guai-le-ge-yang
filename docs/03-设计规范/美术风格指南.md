## ✅ 风格定调：Q 版可爱蒙古风

**关键词**：Q 版、可爱、扁平化、矢量插画、蒙古元素、暖色调

### 风格特征
- **角色比例**：2 ~ 2.5 头身（短粗萌系）
- **羊拐**：保留四面形态特征
- **线条**：圆润、粗描边、无棱角
- **填色**：扁平 + 少量渐变，避免复杂阴影
- **质感**：贴纸/卡通插画，避免写实材质

### 参考方向
- 国内类似画风游戏：《阴阳师》Q 版、《我功夫特牛》、《合成大西瓜》系列
- 文化类似：《蒙古族文化》、《草原民族传统元素》
- 蒙古插画师作品风格（待美术介入时收集 reference）

## 1. 色彩规范

### 1.1 主色调（蒙古传统五色）

| 色彩 | 文化寓意 | 用途 | HEX（参考） |
|---|---|---|---|
| **苍天蓝**（青）| 蒙古族信仰长生天 | 主色，UI 主背景 | `#2E5B8E` / `#5B8FBF` |
| **乳白**（白）| 哈达、奶制品，纯洁吉祥 | 副色，文字底、卡片 | `#F5F1E8` |
| **吉祥红** | 太阳、火、生命 | 强调色、按钮、得分高亮 | `#C53939` |
| **黄金** | 草原日落、佛教尊贵 | 边框、徽章、奖励 | `#D4A044` |
| **草原绿** | 生机、牧场 | 辅色、自然场景 | `#6B9E5F` |

### 1.2 配色原则
- 主色（苍天蓝 + 乳白）占 60%
- 强调色（红 + 金）占 30%
- 辅色（绿）占 10%
- 避免荧光色、纯黑（用深蓝代替）

### 1.3 状态色
- 成功：草原绿
- 警告：吉祥红
- 信息：苍天蓝
- 禁用：暖灰 `#A89F94`

## 2. 字体规范

### 2.1 主字体
- **首选**：思源黑体 / OPPO Sans / 阿里巴巴普惠体（开源可商用）
- 突出"圆润、可爱"调性

### 2.2 特殊场景字体
- **大标题 / Logo**：手写圆体 / 蒙古文装饰字（可用图片形式）
- **数字（得分、计时）**：等宽数字字体（如 DIN Round）

### 2.3 蒙古文（装饰用，可选）
- 配合中文使用时，蒙古文做装饰
- 不要求功能性可读

## 4. 音乐与音效

### 4.1 音乐
- **主旋律乐器**：**马头琴**（试做优先）
- **辅助乐器**：托布秀尔、四胡、口弦琴
- **风格**：宁静、悠远、草原感
- **节奏**：避免过于激昂（除胜利时刻），保留休闲氛围

### 4.2 场景音乐
| 场景 | 音乐风格 |
|---|---|
| 主菜单 | 马头琴轻奏，循环 |
| 弹/抛/抓 进行中 | 节奏感稍强，但不抢戏 |
| 胜利 | 短促欢快，加入长调女声"啊咿"（吉祥音） |
| 失败 | 低沉马头琴一声叹息 |
| "撂四样"特效 | 短笛 + 钟声 + 马蹄声爆发 |

### 4.3 音效
- 弹射：弓弦轻响
- 落地：木碰木的闷响（羊拐相互撞击）
- 翻面：纸张翻动声
- 得分：木鱼/铃声
- 按钮点击：圆润 "boop" 音

### 4.4 是否引入呼麦？
- 默认不用（呼麦深沉，与 Q 版可爱风调性不符）
- 仅在"大师段位"成就解锁时短暂出现 1-2 秒，作为彩蛋

## 6. 关键 UI 元素风格

### 6.1 按钮
- 圆角矩形（圆角半径 = 高度的 20%）
- 描边 2px（深蓝或深红）
- 内部填色 + 文字
- 点击动效：缩小 5% + 阴影下沉

### 6.2 弹窗
- 卡片样式，圆角 20px
- 顶部带蒙古纹饰条
- 半透明遮罩背景（深蓝 + 30% 透明）

### 6.3 图标
- 单色或双色，避免复杂渐变
- 风格统一为"贴纸"风（粗描边 + 圆角）

---

## 7. 动效原则

- **缓动**：以 OutBack / OutElastic 为主，体现"Q弹"
- **时长**：常规 200-300ms，特效 500-800ms
- **节奏**：节制使用，避免眼花
- **粒子**：限制数量（手机性能），用美术贴图替代部分粒子

## 8. 资源规范

### 8.1 分辨率
- 设计基准：750 × 1334（iPhone 8）
- @2x 设计稿，@1x 输出（或矢量为主）

### 8.2 文件格式
| 类型 | 格式 | 说明 |
|---|---|---|
| 静态图 | PNG-8 / WebP | 优先 WebP（压缩比好） |
| 动画 | Spine / 序列帧 | 短动画用序列帧，复杂角色用 Spine |
| 矢量 | SVG / 自绘 Mesh | UI 元素优先矢量 |
| 字体 | TTF（子集化） | 中文必须做子集化 |
| 音频 | OGG / MP3 | OGG 优先（压缩好） |

### 8.3 命名规范
```
ui_btn_play_normal.png       UI 按钮 - 开始 - 默认态
ui_btn_play_pressed.png      UI 按钮 - 开始 - 按下态
char_sheep_idle.spine        角色 - 绵羊 - 待机动画
bg_mainmenu.png              背景 - 主菜单
icon_camel.png               图标 - 骆驼
sfx_throw.ogg                音效 - 抛掷
bgm_main.ogg                 BGM - 主菜单
```
