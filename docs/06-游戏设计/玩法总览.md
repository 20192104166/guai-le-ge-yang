## 一、三玩法定位

三玩法**完全独立**，各自有独立的入口、规则、排行榜、奖励池。玩家从主菜单任选其一。

| 玩法 | 定位 | 核心机制 | 主要考验 | 单局结束条件 |
|---|---|---|---|---|
| **弹沙嘎** | 技巧型 | 弹射 + 碰撞 | 技巧、空间预判 | 一方羊拐输完 |
| **抛沙嘎** | 运气型 | 抛掷 + 落地判定 | 运气、组合识别 | 达目标分 / 输光羊拐 |
| **抓沙嘎** | 挑战型 | 限时摆放 + 抛接 | 反应、手眼协调 | 完成关卡组（如 5 关） |

## 三、单局时长机制（✅ 已确认方案）

### 3.1 弹沙嘎 / 抛沙嘎
- **无固定时长**，按胜负结算（一方羊拐输完 / 一方达目标分）
- **软兜底**：
  - 单回合（一次出手）超 30 秒未操作 → 自动出手（弹/抛默认参数）
  - 单局总时长超 5 分钟 → 弹"加快节奏"提示
  - 单局总时长超 10 分钟 → 强制按当前比分结算

### 3.2 抓沙嘎
- **关卡模式**：单局 = 完成一组 5 关挑战
- **每关有沙包腾空内置时长**（2~5 秒，按难度梯度）
- **整局时长上限**：10 分钟
- **可中途退出**，进度自动保存

## 四、排行榜与社交（✅ 已确认方案）

### 4.1 排行榜结构

```
[ 沙嘎大师榜 ]  ← 综合三玩法加权积分
       │
   ┌───┼─────┬─────┐
   │   │     │     │
[弹]  [抛]  [抓]  [好友榜]
ELO   单局   通关   仅好友
段位  最高   进度   实时
      分     +速度
```

### 4.2 弹沙嘎 - ELO 段位榜
- 段位制（蒙古主题命名）：

| 段位 | 名称 | 说明 |
|---|---|---|
| 1 | 牧羊童 | 入门 |
| 2 | 套马者 | 初级 |
| 3 | 草原骑兵 | 中级 |
| 4 | 巴特尔（勇士） | 高级 |
| 5 | 苍狼之主 | 大师 |
| 6 | 长生天眷顾者 | 巅峰（赛季前 100 名）|

- ELO 算法：标准 ELO 公式，K 值 32
- **赛季制**：每月清零下沉，赛季末发奖励
- 防作弊：仅 **匹配制对战** 计 ELO；好友对战不计

### 4.3 抛沙嘎 - 周榜单局最高分
- 比 **赛季单局最高得分**
- 周榜每周一 0 点清零
- 当周分数 + 历史最高分双展示
- 触发 1000 分 / 撂四样 享额外排名权重

### 4.4 抓沙嘎 - 双榜
- **永久榜**：最高通关数（最远到第几关）
- **周榜**：当周最快通关 5 关用时
- 鼓励"难度突破"与"速度刷新"两种玩法风格

### 4.5 沙嘎大师榜（综合）
综合积分计算：

```
大师分 = (弹 ELO 分 × 0.4) + (抛 历史最高分 × 0.3) + (抓 通关进度积分 × 0.3)
```

- 永久积分
- 季度更新展示榜单

### 4.6 好友榜
- 与抖音好友数据打通
- 显示三玩法各项数据
- 支持"挑战"按钮一键发起异步 PK

## 五、邀请好友 PK 流程

```
玩家 A 在游戏内 → 选择玩法 → "邀请好友"
                              ↓
        生成抖音分享卡片 / 链接（含房间码）
                              ↓
        玩家 B 在抖音收到 → 点击 → 跳转小游戏
                              ↓
                  双方进入异步对战房间
                              ↓
                A 先打完 → 录像存云端 → 通知 B
                              ↓
                B 收到通知 → 进入挑战 → 录像比对
                              ↓
                结算 → 双方都看结果 → 可"再战"
```

> 实时邀请待 E2 确认。

## 六、图鉴系统（待 E6 确认范围）

### 6.1 候选方案
- **a) 仅羊拐皮肤**：内蒙古、外蒙古、青海、新疆、藏族、达斡尔 等地区款式
- **b) 羊拐皮肤 + 文化卡片**：动物、乐器（马头琴/托布秀尔/四胡）、纹样、节日（那达慕/白月）、传说人物（成吉思汗/苏和） 等
- **推荐 b)**

### 6.2 解锁方式
- 玩游戏自然解锁（每达成 N 局 / N 胜送一张）
- 特殊成就解锁（如触发"四骆驼"得金骆驼图鉴）
- 看广告抽卡（单机模式）
- 节日活动限定

## 七、通用元素

### 7.1 羊拐对象
四面四种动物（详见 [`羊拐物理设计.md`](羊拐物理设计.md)）：
- 宽凸面 S = 绵羊
- 宽凹面 G = 山羊
- 窄凸面 H = 骏马
- 窄凹面 C = 骆驼

### 7.2 共享系统
- **形态识别引擎**：羊拐静止后判定其朝上的面（三玩法都用）
- **物理参数**：羊拐的质量、摩擦、弹性等基础参数全局统一
- **皮肤系统**：换装系统，三玩法通用
- **音效库**：共用的弹/落/翻面/计分音效

### 7.3 玩家资产
| 资产 | 说明 |
|---|---|
| 羊拐数量 | 弹/抛沙嘎的"赌注"，输完需要重置（看广告复活或等待） |
| 大师分 | 综合排名货币 |
| 图鉴卡 | 收集型资产 |
| 皮肤 | 装扮型资产 |
| ELO 分 | 弹沙嘎赛季段位 |
