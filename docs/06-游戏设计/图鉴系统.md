## 1. 双轨结构

```
图鉴系统
   │
   ├── 【A 轨】羊拐皮肤图鉴   ← 装扮型，可装备
   │    └── 按地区分类（6 个地区）
   │
   └── 【B 轨】蒙古文化卡片图鉴 ← 收藏型，知识传播
        └── 按类别分类（5 类）
```

## 2. A 轨：羊拐皮肤图鉴

### 2.1 地区分类

| 地区 | 风格特征 | 推荐皮肤数 |
|---|---|---|
| 内蒙古 | 经典蒙古风，4 面贴图融入草原色彩 | 5-8 款 |
| 外蒙古 | 偏古朴，多用赭石、原木色调 | 4-6 款 |
| 青海 | 受藏文化影响，金边 / 转经轮元素 | 3-5 款 |
| 新疆 | 哈萨克 / 维吾尔风格，蓝绿宝石色系 | 3-5 款 |
| 西藏 | 唐卡风，繁复花纹，金箔贴片 | 3-4 款 |
| 达斡尔 | 黑龙江流域，深沉色调 + 桦树纹理 | 2-3 款 |

**首发 V1.0 建议**：每地区 2-3 款，共约 15 款；后续每节日更新 3-5 款。

### 2.2 皮肤稀有度

| 稀有度 | 比例 | 获取难度 |
|---|---|---|
| ⭐ 普通 | 60% | 玩游戏自然解锁（每 5 局送 1 张）|
| ⭐⭐ 罕见 | 25% | 成就解锁 / 看广告抽卡 |
| ⭐⭐⭐ 稀有 | 10% | 段位达成 / 特殊成就 |
| ⭐⭐⭐⭐ 史诗 | 4% | 节日活动限定 |
| ⭐⭐⭐⭐⭐ 传说 | 1% | 单赛季前 N 名 / 触发"四骆驼" |

### 2.3 皮肤数据结构

参考 [`../02-技术文档/服务端架构.md`](../02-技术文档/服务端架构.md) §1.4 `skins` collection。

每款皮肤包含：
- 4 面贴图（绵羊/山羊/马/骆驼面）
- 文化背景描述（200-500 字）
- 解锁条件
- 稀有度

### 2.4 装备规则
- 玩家同时只能装备 1 套皮肤（4 面）
- 装备的皮肤在三玩法中通用
- 切换皮肤无冷却

## 3. B 轨：蒙古文化卡片图鉴

### 3.1 类别

| 类别 | 内容方向 | 首批数量 |
|---|---|---|
| 🐎 动物 | 五畜（绵羊/山羊/马/骆驼/牛）+ 草原野生（雄鹰/苍狼/旱獭/天鹅）| 10-12 |
| 🎵 乐器 | 马头琴、托布秀尔、四胡、口弦琴、长鼓 | 5-6 |
| 🎨 纹样 | 盘长纹、云纹、八宝纹、犄纹、回纹 | 5-7 |
| 🎉 节日 | 那达慕、白月节、祭敖包、马奶节 | 4-6 |
| 📖 传说 | 成吉思汗、苏和与白马、嘎达梅林、长生天 | 4-6 |

**首发 V1.0 总计**：约 30-35 张文化卡片。

### 3.2 卡片数据结构

参考 [`../02-技术文档/服务端架构.md`](../02-技术文档/服务端架构.md) §1.5 `cultureCards` collection。

每张卡片包含：
- 类别 + 名称（中文 + 蒙古文装饰）
- 高清插画（Q 版风格，与皮肤风格一致）
- 长文化描述（300-800 字，介绍文化背景与寓意）
- 稀有度
- 解锁条件

### 3.3 卡片示例

| 卡片 ID | 类别 | 名称 | 简介 |
|---|---|---|---|
| `card_morin_khuur` | 乐器 | 马头琴 | 蒙古族民族乐器之首，琴头雕刻马头 |
| `card_naadam` | 节日 | 那达慕大会 | 蒙古族传统盛会，三项竞技：搏克、赛马、射箭 |
| `card_pattern_panchang` | 纹样 | 盘长纹 | 佛教八宝之一，象征连绵不断、福寿绵延 |
| `card_animal_eagle` | 动物 | 草原雄鹰 | 蒙古猎人之友，象征自由、敏锐、勇敢 |
| `card_legend_suhe` | 传说 | 苏和与白马 | 蒙古族传说，马头琴起源故事 |

### 3.4 解锁机制

| 触发 | 解锁卡片 |
|---|---|
| 首次进入游戏 | 5 张入门卡（绵羊/山羊/马/骆驼/牧民） |
| 完成首局抛沙嘎 | "撂四样"由来卡 |
| 触发"撂四样" | "五畜兴旺"卡（罕见）|
| 触发"四骆驼" | "圣驼"卡（史诗）|
| 解锁任一段位（弹沙嘎）| 对应段位主题卡（如"巴特尔"段位解锁"成吉思汗"卡）|
| 抓沙嘎通关 5 关 | 那达慕节日卡 |
| 单赛季综合大师榜前 10 | "长生天"卡（传说） |
| 节日活动 | 限时主题卡 |
| 看广告（单机模式）| 随机普通卡 |

## 4. 图鉴 UI

### 4.1 主界面
- Tab 切换：羊拐皮肤 / 文化卡片
- 网格布局（4 列 × N 行）
- 未解锁显示为剪影 + 锁标
- 已解锁高亮 + 稀有度光效

### 4.2 详情页
- 大图展示
- 文化背景长文
- 蒙古文装饰（不影响阅读）
- 装备按钮（仅皮肤）
- 分享按钮（分享到抖音 → 解锁成就）

### 4.3 进度提示
- 顶部显示"已解锁 X/Y"
- 按类别 / 地区分组显示完成度
- 集齐一组送特殊奖励（如集齐内蒙古所有皮肤 → 解锁内蒙古限定特效）

## 5. 商业化（不影响平衡）

### 5.1 单机模式抽卡（看广告）
- 每日免费 1 次
- 看激励视频额外 +3 次
- 单次抽卡仅出"普通"和"罕见"
- "稀有"以上必须通过成就 / 段位 / 活动获得（不能买）

### 5.2 后续可选内购
- 节日皮肤礼包（限定，不影响平衡）
- "图鉴加速包"（不直接给卡，给抽卡次数）
- 永远不卖 ⭐⭐⭐⭐⭐ 传说级

## 7. 数据持久化

详见 [`../02-技术文档/服务端架构.md`](../02-技术文档/服务端架构.md) §1.6 `userInventory` collection。

玩家拥有的每张皮肤 / 卡片单独一条记录，便于：
- 增量同步（仅传新增）
- 单卡数据扩展（如收藏次数、装备次数统计）

---

## 8. 文化考据原则（重要）

为保证文化准确性、避免类似"黑龙王"事件：

1. 所有文化卡片内容**必须查证**正规民俗资料
2. 引用来源：
   - 《蒙古族民俗大辞典》
   - 内蒙古博物院公开资料
   - 中国非物质文化遗产网
3. 不确定的内容**不上线**，宁缺毋滥
4. 卡片描述请蒙古族文化顾问审核（如有条件）
5. 用户反馈渠道接受文化错误举报
