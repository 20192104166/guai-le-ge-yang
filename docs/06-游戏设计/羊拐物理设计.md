# 羊拐物理设计

> 本文档定义羊拐（沙嘎）作为物理对象的几何、物理参数与落地判定规则。
> **这是全项目最核心的技术对象，所有玩法都依赖于此。**

---

## 1. 几何形态

羊拐是动物（主要是羊）后蹄踝骨的天然形态，外形近似一个**不规则的近立方体**，四个主要面差异明显。

### 1.1 四面定义

| 面 ID | 名称 | 形态描述 | 对应动物 | 蒙古语 |
|---|---|---|---|---|
| S | 宽凸面（Sheep） | 较宽，外表面凸起 | 绵羊 | Honin |
| G | 宽凹面（Goat） | 较宽，外表面凹陷 | 山羊（或牛） | Yamaa |
| H | 窄凸面（Horse） | 较窄，外表面凸起 | 骏马 | Mor' |
| C | 窄凹面（Camel） | 较窄，外表面凹陷 | 骆驼 | Temee |

> 另两个端面（前后）一般不计入"朝上"形态。如果端面朝上需要重抛。

### 1.2 几何尺寸（参考真实羊拐）

| 参数 | 真实尺寸 | 游戏内尺寸（建议） |
|---|---|---|
| 长度 | 30-40 mm | 1.0 单位 |
| 宽度 | 20-25 mm | 0.6 单位 |
| 高度 | 18-22 mm | 0.5 单位 |
| 重量 | 8-12 g | 由密度推导 |

> 建议在 Phase 1 物理预研阶段，先以网格简化形态（约 200-500 三角形）建模，跑通后再细化。

## 3. 落地概率（核心）

### 3.1 现实统计参考
根据民俗资料与一般经验，单枚羊拐自然抛掷后的概率分布大致为：

| 面 | 期望概率 | 原因 |
|---|---|---|
| 宽凸面（绵羊） | ~ 35-40% | 接触面最大，最稳定 |
| 宽凹面（山羊） | ~ 30-35% | 接触面次大 |
| 窄凸面（骏马） | ~ 12-18% | 较窄，难稳定 |
| 窄凹面（骆驼） | ~ 10-15% | 最罕见，所以"骆驼/马"组合得分高 |

> ⚠️ 上述比例为初步参考值，**需在 Phase 1 通过实测和用户体感校准**。
> 不同地区的羊拐尺寸有差异，最终游戏内的"标准羊拐"概率以游戏设计为准。

### 3.2 实现策略

根据 [`02-技术文档/技术选型.md`](../02-技术文档/技术选型.md) 中的 3D / 2D 选型分两条路：

**A. 3D 真实物理路线**
- 用真实 3D 模型 + 物理引擎自然计算
- 调整质心位置、形状以达到期望概率
- 优点：体验真实，每次掉落都不同
- 缺点：调参成本高，性能消耗大

**B. 2D 加权随机 + 动画路线**
- 物理只控制位置/朝向"看起来对"，最终面通过加权随机决定
- 优点：性能好，结果可控
- 缺点：可能被玩家察觉"假物理"

> 选型决定后，本节会更新具体实现细节。

## 4. 形态判定算法

无论 3D 还是 2D，最终都要回答："这枚羊拐现在哪个面朝上？"

### 4.1 3D 判定
```
向量 up = 世界 Y 轴（0, 1, 0）
对每个面 F：
    法向量 normal_F = 物体局部坐标转世界坐标
    score_F = dot(up, normal_F)
朝上面 = score 最大的 F
若所有 score < 阈值（如 0.7）→ 判定为"侧立"（无效，重抛或自动归位）
```

### 4.2 2D 判定
直接由加权随机结果给出，无需算法判定。

### 4.3 边界情况
- **侧立**：超过 N 秒（如 3s）仍未稳定，判定为侧立，需要"扶正"动画 + 重抛
- **叠在一起**：多枚羊拐叠放时，按各自的朝向独立判定，不互相影响
- **掉出边界**：自动回收，按"重抛"处理

| 玩法 | 用到的物理特性 |
|---|---|
| 弹沙嘎 | 完整刚体物理（碰撞、弹射、摩擦） |
| 抛沙嘎 | 自由落体 + 翻滚 + 形态判定 |
| 抓沙嘎 | 形态判定 + 手动摆位（半物理） |
